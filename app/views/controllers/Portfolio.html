Looking For Group
Angular, Express, PostgreSQL, Node, Bootstrap

A social networking platform on both web-based and ios platforms that allows for tabletop gamers to find local game sessions in their area to join.

This project was built over the course of two weeks, using Node, Express, Postgresql, and BookshelfJS for a back-end api. The web-based application uses Angular 1.5, Bootstrap, and custom SCSS. The ios app was developed using Xcode 7.3 and Swift 2.3, including use of the third party frameworks Alamofire and SwiftyJSON to connect to the API.

Brew Hound
Angular, Express, Node, MapboxGL, Bootstrap

Brew Hound is a web application created in collaboration with Skylar Versage that asks a user's top 3 favorite beers, and locates breweries in the area with recommendations based on their input, along with displaying their location on a map.

This application was built using AngularJS 1.5 and Bootstrap for the front-end, as well as MapboxGL for displaying locations.
